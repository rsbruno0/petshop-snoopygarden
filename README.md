# 🐾 Pet Shop Snoopy Garden

> Uma landing page premium, moderna, responsiva e de alta conversão desenvolvida sob medida para o **Pet Shop Snoopy Garden**, localizado em Salvador - BA.

---

## 🌐 Link de Acesso Público
O site está publicado e pode ser acessado de qualquer dispositivo pelo link do GitHub Pages!

---

## ✨ Principais Funcionalidades

* 📱 **Design 100% Responsivo**: Layout dinâmico que se adapta com máxima perfeição a celulares, tablets e computadores.
* ⚡ **Agendamento Inteligente via WhatsApp**: Formulário prático que valida as informações do tutor e do pet e gera uma mensagem estruturada pronta para envio no WhatsApp.
* 🗺️ **Mapa Interativo em Tempo Real**: Integração com a biblioteca *Leaflet.js* para mostrar a exata localização física do estabelecimento de forma interativa.
* 🧼 **Apresentação de Serviços Essenciais**: Cards modernos com descrições objetivas dos serviços de Banho & Tosa, Consultas Clínicas, Vacinas, Exames, Farmácia e Acessórios.
* 🎨 **Aestética Premium**: Paleta de cores moderna (tons de verde menta, verde floresta e rosa vivo), tipografia de alta legibilidade (Google Fonts) e micro-animações interativas.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estrutura semântica focada em SEO e acessibilidade.
* **Vanilla CSS**: Folhas de estilo customizadas com variáveis globais e flexibilidade total.
* **Vanilla JavaScript**: Efeitos dinâmicos, validação nativa de formulários e integração com APIs.
* **Leaflet.js**: Renderização de mapa interativo de código aberto.

---

## 📁 Estrutura de Arquivos do Projeto

```bash
├── index.html       # Estrutura principal do site e marcação semântica
├── index.css        # Sistema de design, estilos responsivos e animações
├── index.js         # Lógica do mapa, menu mobile e envio de WhatsApp
├── README.md        # Documentação e apresentação do repositório
└── assets/          # Pasta que contém a logo oficial e imagens geradas por IA
```

## 🧼 Como rodar o projeto localmente

1. Faça o download ou clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/petshop-snoopygarden.git
   ```
2. Abra a pasta do projeto e dê um duplo clique no arquivo `index.html` para executá-lo diretamente no seu navegador, ou utilize uma extensão como *Live Server*.
